<Blockquote>BaekJoon Number 1000</Blockquote>
**Problem:** 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.<br>
**Input:** 첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)<br>
**Output:** 첫째 줄에 A+B를 출력한다.

{% highlight c++ %}
#include <iostream>
using namespace std;

int main(){
	int num1, num2;
	
	cin >> num1 >> num2;
	cout << num1 + num2;
	
	return 0;
}
{% endhighlight %}
