<Blockquote>BaekJoon Number 2742</Blockquote>
**Problem:**자연수 N이 주어졌을 때, N부터 1까지 한 줄에 하나씩 출력하는 프로그램첫째 줄에 100,000보다 작거나 같은 자연수 N이 주어진다.<br>
**Output:**첫째 줄부터 N번째 줄 까지 차례대로 출력한다.


{% highlight c++ %}
#include <iostream>

using namespace std;

int main(){
	int num;
	cin >> num;

	for (int i = 0; i < num; i++) {
		cout << num - i << "\n";
	}
}
{% endhighlight %}
