BaekJoon Number 10718<br><br>
Problem: ACM-ICPC 인터넷 예선, Regional, 그리고 World Finals까지 이미 2회씩 진출해버린 kriii는 미련을 버리지 못하고 왠지 모르게 올해에도 파주 World Finals 준비 캠프에 참여했다.<br><br>

대회를 뜰 줄 모르는 지박령 kriii를 위해서 격려의 문구를 출력해주자.


{% highlight c++ %}
#include <iostream>
using namespace std;

int main(){
	for(int i = 0; i <2; i++) {
		cout << "강한친구 대한육군" << endl;
	}
	
}
{% endhighlight %}
